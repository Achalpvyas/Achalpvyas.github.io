Blossom Algorithm

Authors: Achal Vyas and Pruthvi Sanghavi


Summary of the topic in layman’s terms

Blossom Algorithm also known as the Edmunds Algorithm is a matchmaking algorithm. It's an algorithm in graph theory which can be extensively applied to multi agent robotics systems. The pivotal result obtained from this algorithm is maximum matchings can be found out in polynomial time.
